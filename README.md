# PornYami Landing Page

A high-conversion, professional single-page landing page for PornYami - a global adult content API service.

## Features

- 🎨 Modern dark theme with brand colors (Black, Orange, White)
- ⚡ Built with React + Vite for blazing fast performance
- 🎭 Smooth animations with Framer Motion
- 📱 Fully responsive design
- 🔥 Live API sandbox with real-time testing
- 📚 Comprehensive documentation section
- 💰 Clear pricing comparison
- 🎯 Developer-focused UI/UX

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **Axios** - HTTP client
- **Prism.js** - Syntax highlighting

## Getting Started

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Development

The development server will start at `http://localhost:3000`
